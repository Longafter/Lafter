关于网站

  • Github 地址：https://github.com/Longafter
  • 基于 Django2.X & Bootstrap3 开发
  • 数据库使用的是 MySQL
  • 使用 Nginx + Gunicorn 实现将本站部署至阿里云 ECS 服务器上

功能介绍

  • 文章分类，标签，归档，浏览量统计
  • 用户注册，登录，修改个人资料
  • 文章评论，支持 markdown 语法，二级评论结构
  • 消息提醒，登录，注销，收到评论会有消息提示
  • 全文搜索，输入任意关键词即可展示关联文章
  • 后端使用 xadmin 代替 Django 自带的后台管理，界面更加人性化
  • 时间轴，记录网站开发历程
